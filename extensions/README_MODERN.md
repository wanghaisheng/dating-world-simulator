# 现代都市恋爱模拟扩展 (Modern Urban Romance Extension)

> **警告：本模式包含高风险情感博弈内容，请做好心理准备。**

## 📖 简介 (Introduction)

这是“修仙世界模拟器”的一个大型扩展模块，将原本的“修仙证道”核心体验转化为**现代都市背景下的生存与恋爱模拟**。

在这个模式中，你不再是追求长生的修士，而是一名在现代大都会中挣扎求生的普通人。你需要在高压职场与复杂情场之间寻求平衡，在渴望真爱的同时，必须时刻警惕潜伏在亲密关系中的**猎手与陷阱**。

核心玩法从“挂机升级”转变为**“资源管理 + 情感博弈 + 多线攻略”**。

---

## ✨ 核心特性 (Core Features)

### 1. 深度角色模型
每个 NPC 都有独特的现代属性：
*   **社会属性**：职业 (Occupation)、学历、资产。
*   **性格画像**：MBTI 人格类型、穿衣风格、兴趣爱好。
*   **动态状态**：
    *   ⚡ **精力 (Energy)**：每日行动的资源，工作和约会都会消耗。
    *   😨 **压力 (Stress)**：职场内卷和情感修罗场会带来压力，过高会导致崩溃。
    *   😊 **心情 (Mood)**：影响互动的成功率。
*   **隐藏人性 (Hidden Archetypes)**：
    *   NPC 可能隐藏着不为人知的一面，如 **捞女 (Gold Digger)**、**NPD (自恋型)**、**海王 (Player)**、**PUA 控制狂** 等。
    *   你需要通过互动通过 **真实度 (Sincerity)** 属性来判断对方是否真心。

### 2. 拟真社交系统
*   **智能手机界面**：内置“微信”风格的通讯工具。
*   **朋友圈互动**：查看 NPC 的生活动态，点赞或评论。
*   **聊天引擎**：基于 LLM 生成的真实对话，支持撒娇、冷暴力、潜台词识别。
*   **日程系统**：NPC 拥有自己的生活规律（上班、健身、逛街），你可以在特定地点通过**“偶遇”**触发剧情。

### 3. 风险与修罗场 (Risk & Conflict)
恋爱不是童话，是一场博弈。
*   **多线操作风险**：
    *   **安全区**：朋友阶段可以广泛社交。
    *   **危险区**：进入暧昧 (Crush) 阶段后，NPC 会产生排他性。
    *   **修罗场**：若被发现脚踏两只船，将触发毁灭性的修罗场事件，导致社会性死亡。
*   **情感陷阱**：
    *   遭遇“杀猪盘”导致资产清零。
    *   被 PUA 导致自信心永久下降。
    *   被 NPD 长期精神打压导致抑郁。

## 🚀 快速开始 (Getting Started)

### 1. 切换游戏模式
在 `src/config.py` 或 `static/config.yml` 中将游戏模式设置为 `MODERN_ROMANCE`。

```python
# src/config.py
GAME_MODE = GameMode.MODERN_ROMANCE
```

### 2. 启动游戏
运行标准启动命令：
```bash
python src/server/main.py --dev
```

### 3. 体验流程
1.  **创建角色**：选择现代身份（如程序员、金融分析师、偶像练习生）。
2.  **每日循环**：规划每日行程（工作赚钱 vs 约会花钱）。
3.  **社交互动**：使用手机添加 NPC 联系方式，发起聊天或邀约。

## 🏗️ 技术架构 (Architecture)

本扩展在保留原有的 ECS (Entity-Component-System) 架构基础上进行了模块化扩展：

*   **数据层**:
    *   `ModernProfileMixin`: 扩展 `Avatar` 类，注入现代属性。
    *   `RelationshipStage`: 定义 6 级关系阶段（陌生人 -> 伴侣）。
*   **系统层**:
    *   `SocialSystem`: 处理聊天、约会逻辑。
    *   `ScheduleGenerator`: 基于职业生成 NPC 每日行为树。
    *   `RiskEngine`: 每日检测情感风险和多线冲突。
*   **表现层**:
    *   前端新增 `PhonePanel` (手机组件) 和 `ModernStats` (现代状态栏)。
